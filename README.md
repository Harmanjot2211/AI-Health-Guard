# AI-Health-Guard
**AI HealthGuard – Smart Disease Detection System** is an AI-powered web application that helps users gain early health insights by analyzing their symptoms and predicting possible diseases. The platform allows users to enter symptoms by typing, selecting from a searchable list, or using voice input. Multiple symptoms can be added easily, and the system analyzes them using an AI prediction engine.

After analysis, the application displays the top possible diseases along with probability percentages, visual progress bars, and brief descriptions. The system may also ask follow-up questions and consider optional health details such as age, gender, allergies, or existing conditions to improve accuracy.

AI HealthGuard also includes a risk level indicator, emergency alerts for critical symptoms, a health dashboard, and recommendations for next steps. The interface features a modern medical design with responsive layout and dark/light mode support. The tool provides preliminary health insights and encourages users to consult healthcare professionals for medical advice.
